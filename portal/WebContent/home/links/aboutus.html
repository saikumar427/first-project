Company
 

The First Indian Community Portal built using Web 2.0 technologies such as Blogging, Social Networking, Photo Sharing and RSS Feeds. It is our mission to bring you most innovative community networking technologies, and provide extensive customization tools that enable you to express yourself the way you want it!

Eventbee's community features include creating local special interest groups, friends networking, event management, classifieds, volunteering. Eventbee constantly monitors community networking technologies and rolls out new features that are beneficial to the community.

We proudly use open source software from Apache, Postgre Sql, My Sql, Open Ldap, Qmail, Sendmail, Jboss, Linux and Java from Sun. We thank open source community for providing these technologies free-of-cost.

Eventbee gives back portion of advertisement revenues towards drinking water problem in India. Eventbee closely works with other Indian Non-profits in achieving this goal.
