Network Ticket Selling Options

Partners promote your events in two ways

1. Web Promotion

Partners copy and paste custom URL links and buttons on their web site or blog. Various settings under this option allows you to control how Partner can sell tickets from the Web.

Default settings under this section:

* 100% commission to Partner - Here Partner receives all the commission set by you on each Ticket sale.

* Deny permission to change commission settings by Partner - By default Partner is not allowed to change commission settings or pass commission to the Attendee. You can change this default setting to allow Partner to change commission settings.

2. Social Promotion

Partners promote events to their Social Graph by publishing event feeds to Facebook profile, inviting friends with embedded custom URL links etc. Various settings under this option allows you to control how the Partner can sell tickets using this feature.

Default settings under this section:

* 50-50% commission split between Partner and Attendee - Here Partner receives 50% of the commission as earnings and Attendee receives 50% of commission as a discount towards event registration.

* Deny permission to change commission settings by Partner - By default Partner is not allowed to change commission settings. However you can allow Partner to change commission settings.

NOTE: These are default global settings for all Partners, having said that, you can modify these default settings on any Partner case by case basis.
